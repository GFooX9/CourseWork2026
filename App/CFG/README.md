# 📁 Модуль конфигурации (Configuration Layer)

## 📌 Назначение
Централизованное управление путями файловой системы, ресурсами и системными вызовами ОС. Обеспечивает сквозную стабильность приложения как при локальном запуске, так и при развертывании внутри скомпилированного бинарного пакета `.exe`.

## ⚙️ Состав и зоны ответственности
* **`config.py`** — Глобальный конфигуратор. Содержит функцию `resource_path()`, которая перехватывает переменную окружения PyInstaller `sys._MEIPASS` для работы с кэшем временных файлов сборщика. 

## 🔒 Правила рефакторинга
Хардкод абсолютных (`C:/...`) или жестких относительных путей внутри бизнес-логики **запрещен**. Любой новый ассет (иконка, шрифт, модель) обязан регистрироваться в этом модуле через обертку `resource_path()`.
