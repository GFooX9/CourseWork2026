# 📁 Модуль генерации отчетности (Reporting Layer)

## 📌 Назначение
Автоматизированное формирование официальных экспертных заключений и выгрузка результатов ИИ-исследований в кроссплатформенный формат печати.

## ⚙️ Состав и зоны ответственности
* **`report_generator.py`** — Движок компиляции документов на базе **ReportLab**. Переведен на конвейерную автоматическую верстку `SimpleDocTemplate` с динамическим переносом длинных заметок на новые страницы. Отвечает за регистрацию кириллического шрифта `NotoSans`, внедрение защитных двухстрочных подвалов и интеграцию логотипа-печати в шапку хедера через невидимые таблицы `Table`.

## 🔒 Правила рефакторинга
Использование жестких низкоуровневых координатных сеток (`canvas.Canvas` для больших блоков текста) не допускается из-за риска обрезания длинных строк. Любой текстовый абзац должен оборачиваться в потоковый элемент `Paragraph`.
