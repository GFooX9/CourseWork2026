# 📁 Графический интерфейс пользователя (Presentation Layer)

## 📌 Назначение
Рендеринг визуальных элементов, обработка пользовательского ввода (клик, слайдер, drag-and-drop) и отображение результатов вычислений ядра ИИ.

## ⚙️ Состав и зоны ответственности
* **`main_window.py`** — Корневой каркас. Управляет глобальной палитрой темы оформления **Dark Lavender**, координирует стек экранов `QStackedWidget` и распределяет радио-подсветку активных вкладок.
* **`screens/`** — Полноэкранные рабочие пространства. `identifier_screen.py` отвечает за интерактивную панель настроек (сиреневый слайдер точности, скруглённый крупный чекбокс).
* **`components/`** — Диалоговые и вспомогательные окна (`ObjectEditorDialog`, `ProtocolDialog`, `YoloInstallerDialog`). Содержит QSS-стили полупрозрачного стеклянного дизайна `rgba` с мягкими неоновыми контурами и кастомными сиренево-серыми дорожками полос прокрутки.

## 🔒 Правила рефакторинга
Тяжелые вычисления (инференс нейросетей, сетевые запросы `requests`) внутри этого слоя **категорически запрещены**. Любая ресурсоемкая задача обязана выноситься в фоновые потоки `QThread` (например, `AnalysisWorker`), общаясь с UI исключительно через механизм сигналов `pyqtSignal` и слотов `pyqtSlot`. Это защищает графическую оболочку от фризов.
